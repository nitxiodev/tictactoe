# tictactoe
Tic-tac-toe game using minimax algorithm with alpha-beta pruning based on Russell and Norvig pseudocode.

## Instructions
To run the tic-tac-toe game, simply type this command in the shell:

```python
python game.py
```

### Board game

```bash
| (0) O | (1) O | (2) X |
| (3) X | (4) X | (5) O |
| (6) O | (7) X | (8) X |
```
